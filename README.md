## 🎮 WHAT'S INSIDE

> A full number genius arcade — 8 games + a scientific calculator, all wrapped in a glowing Minecraft-blue world built just for Jaxon.

| # | 🎮 Game / Feature | 📋 Description |
|:---:|:---|:---|
| 1 | 🧮 **Calculator Lab** | Scientific calculator that **speaks every number and operator out loud** |
| 2 | 🎯 **Prime Hunter** | Race the clock to click every prime number on the board |
| 3 | 🌀 **Fibonacci Builder** | Complete missing numbers in the Fibonacci sequence |
| 4 | 💻 **Base Converter** | Convert numbers between Binary, Octal, Decimal and Hex |
