# Jaxon's Number Genius Lab

An alternate blue, Minecraft-inspired number arcade for Jaxon.

This repo is intentionally separate from `Jaxons-Numbers-Games`. It uses the same broad idea, but a different layout and implementation.

## Features

- Big breathing title
- Floating, blinking numbers
- Cursor-follow number sparks
- Voice toggle with animated voice bars
- Speech for calculator input, game intros, answers, and feedback
- QR station for phone or tablet access
- Calculator Lab
- Prime Hunter
- Fibonacci Builder
- Base Converter
- Multiplication Master
- Magic Square
- Factorial Blast
- Pattern Detective
